# Automation Readiness Checklist

## Process Clarity
- [ ] Process steps are documented
- [ ] Inputs and outputs are defined
- [ ] Owners are assigned for each stage

## Data Readiness
- [ ] Source data is consistent
- [ ] Data quality issues are known
- [ ] Required data fields are available

## Tooling Context
- [ ] Existing systems are identified
- [ ] API/access constraints are known
- [ ] Security approvals are understood

## Business Case
- [ ] Time/cost baseline is measured
- [ ] Automation target metric is set
- [ ] Priority process selected

## Change Management
- [ ] Team training needs identified
- [ ] Fallback/manual process defined
- [ ] Monitoring and alerting approach planned

## Launch Readiness
- [ ] Pilot scope selected
- [ ] Acceptance criteria agreed
- [ ] Post-launch review date set
