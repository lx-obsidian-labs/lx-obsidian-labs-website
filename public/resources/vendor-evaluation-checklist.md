# Vendor Evaluation Checklist

Use this checklist to evaluate software and systems partners.

## Strategy Fit
- [ ] Understands our business goals
- [ ] Proposes measurable outcomes
- [ ] Can explain trade-offs clearly

## Technical Capability
- [ ] Relevant stack experience
- [ ] Integration capability
- [ ] Security and data handling clarity
- [ ] Scalable architecture approach

## Delivery Quality
- [ ] Clear implementation phases
- [ ] Milestone-based delivery plan
- [ ] QA and testing process defined
- [ ] Deployment and handoff support included

## Communication
- [ ] Weekly update rhythm
- [ ] Named project lead
- [ ] Risk escalation process

## Commercials
- [ ] Scope and assumptions documented
- [ ] Pricing model is transparent
- [ ] Change request process defined

## Trust Signals
- [ ] Comparable case studies
- [ ] Client references available
- [ ] Long-term support options
